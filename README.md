# Winter World Clone

A responsive front-end clone of the **Winter World Pakistan** website, built to practice modern HTML and CSS techniques. This project recreates the overall layout, styling, and responsive behavior of the original website while focusing on clean code structure and reusable components.

> **Disclaimer:** This is an independent front-end recreation of the Winter World Pakistan website. All trademarks, logos, images, and brand assets belong to their respective owners.

## Features

* Responsive desktop, tablet, and mobile layouts
* Hero section with responsive background images
* Navigation bar with mobile hamburger menu
* Explore cards for different attractions
* Flexible gallery layout using Flexbox
* Floating shopping cart button
* Multi-column footer with useful links and timings
* Clean semantic HTML structure
* Organized CSS with reusable utility classes
* Mobile-first responsive adjustments using media queries


## Built With

* HTML5

* Media Queries
